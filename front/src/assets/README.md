### 7. Добавление импортов для графических ассетов

Создадим папку для ассетов и добавим в нее необходимые файлы (например, иконки и логотипы):

# Assets Directory

This directory contains all the static assets used in the EventPortal application.

## Contents:
- Images for the application
- Quiz icons
- Event placeholder images 
- Other static resources

## Usage:

All files can be referenced in templates using:

```html
<img src="assets/filename.ext" alt="Description">
```
